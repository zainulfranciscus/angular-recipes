This page demonstrates the difference between root scope and a scope. A root scope is a scope created on the HTML elements that contains the ng-app directive. A root scope is available in the entire application.

In this page, a variable color exist both in the root scope and controller's scope. This application display color which exist in the current scope.

The rootScope's favorite color:

{{color}}

The scope of the controller's favorite color:

{{color}}

The rootScope's favorite color is still:

{{color}}

Notice that controller's color variable does not overwrite the rootScope's color value.

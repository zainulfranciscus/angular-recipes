The purpose of this script is to demonstrate various built-in filters in Angular JS. such as lower case, currency, and filter. This page also demonstrates a custom filter named 'capitalizeFirstLetter', which is constructed using a filter factory.

Name (capitalize first letter) Country (lower case) Income (with 3 fractions)
{{x.name | capitalizeFirstLetter}} {{x.country | lowercase}} {{x.income | currency : x.currency : 3}}
